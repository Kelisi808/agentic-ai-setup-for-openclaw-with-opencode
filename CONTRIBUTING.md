# Contributing

Thank you for improving this setup repository.

## How to contribute
1. Open an issue describing the problem or enhancement.
2. Fork the repository and create a focused branch.
3. Keep instructions beginner-friendly and copy/paste ready.
4. Submit a pull request with clear before/after details.

## Contribution standards
- Keep commands tested on the target platform.
- Avoid committing secrets, tokens, or private keys.
- Prefer simple language over jargon.
- Keep file names descriptive and platform clear.

## Good first contributions
- Add troubleshooting fixes for common setup errors.
- Improve command validation steps.
- Add short examples for real AI agent workflows.
- Improve PitStop reliability and operations guidance.
