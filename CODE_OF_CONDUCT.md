# Code of conduct

We want this project to be welcoming, safe, and useful for people everywhere.

## Our commitment
As contributors and maintainers, we commit to respectful, constructive, and inclusive collaboration.

## Expected behavior
- Be respectful and patient.
- Assume good intent and ask clarifying questions.
- Give actionable feedback focused on the work.
- Share knowledge in beginner-friendly language.
- Protect user safety, privacy, and security.

## Unacceptable behavior
- Harassment, hate speech, discrimination, or personal attacks.
- Doxxing, stalking, or intimidation.
- Sexualized language or unwanted sexual attention.
- Deliberate sharing of secrets, credentials, or exploit details in public.
- Disruptive behavior that blocks collaboration.

## Scope
This code of conduct applies to:
- repository issues, pull requests, and discussions
- project-related communication spaces

## Enforcement
Maintainers may remove, edit, or reject contributions and comments that violate this code.
Repeated or severe violations may lead to temporary or permanent bans from project spaces.

## Reporting
Report conduct concerns through private channels when possible (e.g., maintainers or GitHub moderation/reporting tools).
Include links/screenshots/context so maintainers can review fairly.

## Attribution
This policy is informed by the Contributor Covenant (v2.1) and adapted for this repository.
